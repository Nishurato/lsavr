# Speedrun Autorecord Tool

This is a script for OBS working with LiveSplit commands. It still in development state.

![SAR-tool demonstation](/media/demo.gif)

## Instalation

1. Download auto_record.py or clone repository.
2. Open Script dialog window (Tools > Scripts).
3. Click + button and locate auto_record.py.
4. Done!
