# Speedrun Autorecord Tool

This is a script for OBS working with LiveSplit commands. It still in development state.

![SAR-tool demonstation](/media/demo.gif)

## Instalation

1. Download auto_record.py or clone repository.
2. Install Python and required libraries (commands included in requirements.txt)
3. Open Script dialog window (Tools > Scripts).
4. Click + button and locate auto_record.py.
5. Start TCP Server in LiveSplit (Control > Start TCP Server)
6. Done!

## Requirements

- Python 3.12
- Libraries: psutil (obspython included in OBS)
